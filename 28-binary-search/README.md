## What It Does

Implements binary search with iterative and recursive approaches plus a Streamlit visualization showing each step.

## Run It

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Core Concepts

- Iterative binary search
- Recursive binary search
- Step-by-step search visualization
- Time complexity comparison with linear search
- Streamlit interactive UI

## What You Will Learn

You will learn how binary search achieves O(log n) performance and the difference between iterative and recursive approaches.
