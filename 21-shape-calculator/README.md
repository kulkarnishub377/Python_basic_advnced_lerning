## What It Does

Calculates areas and perimeters for geometric shapes using an OOP inheritance hierarchy.

## Run It

```bash
python main.py
```

## Core Concepts

- Abstract base classes and inheritance
- Method overriding for shape calculations
- Polymorphism in action
- Mathematical geometry formulas
- Clean class hierarchy design

## What You Will Learn

You will learn inheritance, polymorphism, and how abstract classes enable extensible design.
