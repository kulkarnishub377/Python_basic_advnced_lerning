## What It Does

A fully interactive command-line calculator that performs addition, subtraction, multiplication, and division. Includes robust error handling for invalid inputs and division by zero.

## Run It

```bash
python calculator.py
```

## Core Concepts

- Defining and calling functions for each arithmetic operation
- Handling `ZeroDivisionError` exceptions
- Using `while True` loops for continuous operation
- Clean user prompts and input parsing
- Modular function design

## What You Will Learn

You will understand how to structure a program using functions, handle runtime errors, and build a tool that runs continuously until the user decides to quit.
