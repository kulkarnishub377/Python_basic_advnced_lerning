🛠️ Auto-Generated CMS

This project uses the `sqladmin` package. It reads your raw backend SQLAlchemy classes, and instantly renders a full graphical interface for you to manage the database without writing a single line of frontend code.

Launch SQLAdmin Dashboard
How it works:
  • No custom API routes needed for CRUD.
  • Fully handles foreign-key relationships (Categories → Posts).
  • Built-in search, pagination, and sorting.
