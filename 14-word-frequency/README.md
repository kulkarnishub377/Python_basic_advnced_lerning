## What It Does

Analyzes text files to count word frequencies with sorting options and stopword filtering.

## Run It

```bash
python counter.py
```

## Core Concepts

- Dictionary-based frequency counting
- File reading and text processing
- Sorting with `sorted()` and lambda
- String normalization and tokenization
- Formatted table output

## What You Will Learn

You will learn text processing including tokenization, frequency analysis, and dictionary-based counting.
