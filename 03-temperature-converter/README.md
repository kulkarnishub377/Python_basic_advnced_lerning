## What It Does

Converts temperatures between Celsius, Fahrenheit, and Kelvin scales. Supports bidirectional conversion with a menu-driven interface and formatted output.

## Run It

```bash
python converter.py
```

## Core Concepts

- Mathematical conversion formulas between temperature scales
- Formatted string output with f-strings
- Menu-driven program flow
- Input validation and error handling
- Continuous loop until user exits

## What You Will Learn

You will learn formula-based computation, clean output formatting with f-strings, and building user-friendly menu systems.
