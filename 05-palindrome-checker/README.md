## What It Does

Checks whether a given word, phrase, or sentence is a palindrome. Handles spaces, punctuation, and case-insensitivity for robust checking.

## Run It

```bash
python palindrome.py
```

## Core Concepts

- String slicing with `[::-1]` for reversal
- String normalization with `.lower()` and `.replace()`
- Boolean logic and conditional checks
- Regular expressions for filtering
- Interactive input loop

## What You Will Learn

You will learn string manipulation techniques including reversal, normalization, and filtering.
