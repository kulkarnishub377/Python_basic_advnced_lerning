## What It Does

Reads CSV files and generates formatted summary reports with filtering, sorting, and basic statistical analysis.

## Run It

```bash
python main.py
```

## Core Concepts

- Reading CSV files with the `csv` module
- Data filtering and sorting
- Generating formatted text reports
- Working with tabular data
- File path handling

## What You Will Learn

You will learn how to process structured data files, extract statistics, and generate human-readable reports.
