## What It Does

Implements a hash table from scratch with separate chaining for collision resolution and dynamic resizing.

## Run It

```bash
python hash_table.py
```

## Core Concepts

- Custom hash function
- Separate chaining collision resolution
- Dynamic resizing for load factor
- Key-value storage and retrieval
- Hash distribution understanding

## What You Will Learn

You will learn how hash tables achieve O(1) lookups, how collision resolution works, and why they underpin dictionaries.
