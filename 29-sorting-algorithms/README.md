## What It Does

Implements Bubble Sort, Merge Sort, and Quick Sort from scratch with step counts and performance comparisons.

## Run It

```bash
python app.py
```

## Core Concepts

- Bubble Sort with early termination
- Merge Sort with divide-and-conquer
- Quick Sort with partition logic
- Step counting for complexity analysis
- Modular structure (`bubble.py`, `merge.py`, `quick.py`)

## What You Will Learn

You will learn how sorting algorithms work internally and when to choose one over another.
