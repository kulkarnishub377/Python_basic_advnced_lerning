## What It Does

The classic FizzBuzz programming challenge. Prints numbers from 1 to N, replacing multiples of 3 with 'Fizz', multiples of 5 with 'Buzz', and multiples of both with 'FizzBuzz'.

## Run It

```bash
python fizzbuzz.py
```

## Core Concepts

- Modulo operator (`%`) for divisibility testing
- Conditional logic with `if/elif/else` chains
- Range-based `for` loops
- Output formatting and printing patterns
- Classic interview problem

## What You Will Learn

You will understand conditional ordering, how modulo arithmetic works, and why FizzBuzz is a foundational coding exercise.
