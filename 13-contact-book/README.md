## What It Does

A persistent contact management system with JSON storage, supporting search, add, edit, and delete operations for contacts with name, phone, and email.

## Run It

```bash
python contacts.py
```

## Core Concepts

- JSON-based data persistence
- Search with string matching
- CRUD operations on contact data
- Data validation for phone and email
- Table-formatted display

## What You Will Learn

You will learn how to build a complete CRUD application with file-based storage and search functionality.
