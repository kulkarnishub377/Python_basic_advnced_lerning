## What It Does

Simulates a bank account system with deposit, withdrawal, and balance inquiry operations. Demonstrates class-based design and state management.

## Run It

```bash
python bank.py
```

## Core Concepts

- Class-based design with `__init__`, `deposit`, `withdraw`
- Instance variable state management
- Validation for insufficient funds
- Transaction history tracking
- Formatted financial output

## What You Will Learn

You will learn OOP fundamentals with classes, managing object state, and validating business rules.
