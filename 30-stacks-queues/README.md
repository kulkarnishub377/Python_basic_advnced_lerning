## What It Does

Implements Stack (LIFO) and Queue (FIFO) data structures from scratch with push, pop, peek, and overflow handling.

## Run It

```bash
python app.py
```

## Core Concepts

- Stack with `push`, `pop`, `peek`
- Queue with `enqueue`, `dequeue`
- LIFO vs FIFO ordering
- Overflow and underflow handling
- Class-based data structure design

## What You Will Learn

You will learn how stacks and queues work internally and their use cases in real applications.
